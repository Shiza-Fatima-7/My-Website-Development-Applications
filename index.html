HTML, or HyperText Markup Language, is the standard backbone used to create and structure content on the World Wide Web. Rather than a programming language that handles logic, HTML is a markup language; it uses a system of "tags" to tell a web browser how to display text, images, and interactive forms. HTML was conceived in 1989 by Tim Berners-Lee, a physicist at CERN (the European Organization for Nuclear Research). At the time, researchers struggled to share information across different computer systems.
